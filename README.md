Adventure Game – The Last Crystal of Eldoria

Where the code is hosted:
GitHub repository

External services:
PythonAnywhere (for running the game)

Languages and technologies:
Python 3

System requirements:
Python 3 installed

Coding conventions:
- Each chapter is in a separate file (chapter1–chapter5)
- main.py controls the game flow
- player.py contains the Player class

How to run the program:
Run the following command:
python3 main.py

Architecture overview:
- main.py → controls game flow
- chapter files → game logic
- player.py → stores player data

How to start the game:
Run main.py and follow the instructions in the console.
